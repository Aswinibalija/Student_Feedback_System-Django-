# 🎓 Student Feedback System (Django)

A simple Django project to collect student feedback with fields like student name, email, course, rating, and feedback message. After submission, it stores feedback in the database and displays a thank-you confirmation.

---

## 📂 Project Details
- **Project Name:** `student_feedback`
- **App Name:** `feedback`
- **Framework:** Django (Python)
- **Database:** SQLite (default)

---

## 🛠️ Features
✅ Collect student feedback with fields:
  - Student Name  
  - Email  
  - Course  
  - Rating (1-5)  
  - Feedback Message  
  - Timestamp (auto-generated)

✅ Save feedback to the database  
✅ Display a "Thank You" confirmation page after submission
